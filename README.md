# 지하철 최단거리를 알려주는 프로그램

출발역과 도착역을 입력시 최단거리(출발역-(환승역)-도착역)와 배차시간을 고려한 총 소요시간을 알려주는 프로그램입니다.  
역 데이터는 부산 지하철 노선도를 사용하였습니다

Released in 2019-06-10

Collaborator 안지민, 김지훈
